## Утверждено пользователем

| Тема | Решение |
|------|---------|
| Название бренда | **Smailee** (латиница) в шапке/лого |
| Стек | Next.js (App Router) + TypeScript |
| Стилизация | **Tailwind CSS** |
| Язык интерфейса | Русский везде (лендинг + ЛК) |
| Цвет бренда | **Мятный (основной) + Indigo (акцент)** на светлом фоне |
| Локальная БД | PostgreSQL 16 (Homebrew), БД `smailee_dev`, юзер `zaytsegor` |
| ORM | Prisma (провайдер postgresql) |
| Авторизация | Своя, email + пароль (хэш bcrypt, сессия через JWT в httpOnly cookie) |
| Провайдер email | Unisender Go (адаптер, пока заглушка) |
| LLM | Claude / Anthropic (адаптер, пока заглушка) |
| CRM | Битрикс24 (адаптер, пока заглушка) |
| Заявки с лендинга | В БД (таблица `leads`) + экспорт в Excel из админки |
| Контакты в футере | Плейсхолдеры (hello@smailee.ru, @smailee) |
| Генерация картинок | fal.ai, модель **Recraft v3** (`digital_illustration`) — даёт чёткие flat-иллюстрации. FLUX не используем (размывает). **ЛИМИТ: не более 50 генераций всего** |
| Стиль картинок | Флет-иллюстрации, мятный+indigo, светлый фон, европейская внешность людей (ру-регион) |
| Приоритет за ночь | 1) Лендинг полностью с картинками 2) каркас ЛК/MVP с заглушками |

## Принцип работы с внешними API
- Все внешние сервисы (Unisender, Anthropic, Bitrix24) обёрнуты в **адаптеры** с единым
  интерфейсом. Пока ключей нет — адаптер работает в mock-режиме (детерминированные
  фейковые данные), но НЕ ломает сценарий.
- Как только пользователь впишет ключи в `.env` — те же адаптеры переключаются на реальные
  вызовы **без изменения кода вызывающей стороны** (проверка `if (process.env.KEY) {...}`).
- Внутренняя логика (БД, очередь отправки, статусы писем, дашборд) — **настоящая**, не заглушка.

## Готовность к проду (Amvera)
- Все секреты — только через переменные окружения, ничего в коде.
- `Dockerfile` + инструкция, чтобы перенести на Amvera.
- БД переключается сменой `DATABASE_URL` (локальный Postgres → прод Postgres).
- Никаких абсолютных путей, нет зависимости от локальной ФС для рантайма (кроме кэша картинок в public).

## Лимит генераций картинок (бюджет 50)
План расходования (~11-14 шт.):
- Логотип: 1 (✅ уже есть `logo-test.png`)
- Hero-иллюстрация (человек чиллит, письма летят сами): 1-2
- Иллюстрации для блока «Боли»: 1
- Иллюстрации для 4 features: до 4
- Иконки-иллюстрации для шагов: до 3
- Запас на переделки: остальное
